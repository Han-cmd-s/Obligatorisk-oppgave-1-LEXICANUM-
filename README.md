Tilbakemledings Notater for fremtidige endringer: 
Lurt å gi brukeren en suksessmelding etter handlinger (f.eks opprettelse av kurs eller bok).
Endre datatype år i Bok - lag det som en int med validering.
ISBN burde ha validering - brukeren kan legge inn 1 eller 5 tall, men ISBN har fast lengde.
Endre logikk slik at systemet ikke tillater negativt antall bøker ved registrering.
Vurdere å endre try-catch logikken i menyen - det kan være bedre med mindre blokker eller TryParse.
Gjøre feilmeldingene mer konsistente og forklarende. 
Klassenavn bør være tydelige og beskrivende, så det er klart hva de representerer.
